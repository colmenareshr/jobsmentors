## Description: <br>
Bind pre-downloaded Jetson reference docs (developer guide, design guide, pinmux, schematics) into the active profile documents block. <br>

This skill is ready for commercial/non-commercial use. <br>

Developers and engineers use this skill to register paths for pre-downloaded Jetson reference documents into a target-platform profile so downstream BSP customization skills can resolve document paths by name. <br>
